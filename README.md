### description

My Portfolio built with React.

### Features

- Fully Responsive
- Multi-Page Layout
- Contact Form With EmailJs
- React-Bootstrap
- Edit Content From One Place

### Setup

Get the code.

Install required dependencies

<pre>yarn install</pre>

modify pages content in `src/content_option.js`.

start the server

<pre>yarn start</pre>
